# Size Scanner

A fast, lightweight Windows tool that scans every drive and shows you the biggest files and folders eating up your space. Sort, reveal in Explorer, or send space hogs straight to the Recycle Bin. Built with Tauri.

## What it does

- Scan all drives for the largest **files** or the largest **folders**.
- Pick how many results to show (top 50 by default, up to 500).
- Pause, resume, or stop a scan at any time.
- Select files and move them to the Recycle Bin, or reveal them in Explorer.
- Right-click any result to copy its path or open it in Explorer.
- Export the list as CSV or TXT.
